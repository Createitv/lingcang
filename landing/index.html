小红书工作台

一个窗口，多个干净身份。

每个账号使用独立持久会话，保留登录状态和页面位置，不需要反复退出、扫码、切浏览器。

会话隔离

每个账号有独立的 Electron partition，Cookie、登录态和权限策略互不串扰。

跨平台发布

macOS、Windows、Linux 都由 GitHub Actions 自动打包并发布到 Release。

轻量桌面端

侧栏、标签、地址栏都围绕高频运营动作设计，少解释，多操作。
